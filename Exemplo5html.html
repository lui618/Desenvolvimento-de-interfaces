exemplo 5 HTML - tags de cessçao e títulos

Este exemplo mostra como usar as tags de título >h1> à >h6> para organizar o conteúdo do documento em cessões hierárquicas.

Descrição

As tags de título são usadas para definir cabeçalhos e subtítulos. Elas variam de >h1>(mais importante) à >h6> (menos importante).
