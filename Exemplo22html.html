Exemplo 26 CSS - Tipos de Vinculação de arquivos CSS e sistemas de cores

Existem 3 tipos básicos de vinculação de arquivos CSS, são elas: as vinculações externas e as internas que se dividem em duas incorpodas e inline.

Neste exemplo, nós estamos utilizando a vinculação externa, que é a mais indicada para projetos maiores

Para vincular arquivos CSS de forma externa, nós usamos a tag link.

Estilos de Fonte

Em css os estilos de fontes sao amplamentes empregados em textos para torna-los mais atrativos e profissionais.

Os estilos de fontes em css podem variar de diferentes formas com diferentes aplicaçoes que vao desde mudanças de cores familia,estilo,peso dentre outras opçoes.

Cores e estilos de fundo

Uso do background com sistemas de cores RGB e RGBA

Sistema de cores RGB

O sistema de cores RGB representa um esquema de cores baseado nas cores R(veremlho) G(verde) B(azul),cada perimetro do rgb define a intensidade da cor e ela varia de 0 a 255. Para exemplo rgb(255,99,71)

Sistema de cores RGBA

O sistema de cores rgba sao uma extensao ds valores rgb com um canal alfa (a),que especifica a opacidade que pode variar de 0 a 1 sendo o 0 totalmente transparente e o 1 totalmente opaco.Por Exemplo rgba(35,99,71,0.3
