Exemplo 4 Html - estrutura geral de um docmunento HTML

Este exemplo mostra a estrutura mínima de um documento html. Aqui estão as príncipais tags utilizadas:

  • <!DOCTYPE html >: Define o tiopo de documento html, ou seja a versão do html que será usada na página.
  • < html >: A tag html é o elemnto principal do documento html, ela define o container raiz de toda pagina.
  • < HEAD>: A tag head contém os metadados que serao interpretados pelo navegador. Nela encontaremos o título da página e links e arquivos para configuração do documento
  • < body >:A tag body e responsavel pela exibição do conteudo visivel do documento, ou seja, tudo que estiver neste bloco será visivel para o usuário.
