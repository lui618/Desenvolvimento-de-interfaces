HTML - Web Semântica

Neste bloco estamos usando a tag semântica <section> para definir uma seção genérica de um documento.

A web semântica refere-se ao uso do HTML e outras tecnologias para estruturar o conteúdo de uma página de froma que ela seja compreendida tanto por humanos quanto máquinas. Isso significa usar elementos HTML que descrevam o significado do conteúdo, em vez de apenas sua aparência.
