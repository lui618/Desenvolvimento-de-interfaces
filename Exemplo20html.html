Visao geral sobre Css

Css é um acronimo para folhas de estilo em cascata, essa tecnica visa personalizar estilizar um documento html, para isso usamos regras css que representam uma estrutura que define como o slementos selecionados devem ser estilizads. Essa regra é composta pelas seguintes partes

  • Seletor:Seletor indica quais elementos serao afetados pelo estilo definido
  • Bloco de declaração:é o bloco composto por propriedades e valores , esse bloco será aplicado ao elemento html.

Tipos de seletores

Em css seletroes sao padroes usados para selecionar e estilizar os elementos html em uma pagina web.Eles definem quais elementos receberão as regras de estilos que serão declaradas dentro das chaves.Possuimos varios tipos de seletores os principais são:

Seletores de ID

O seletor de id é aquele que aplica a regra css a um componente html que tenha um atributo id no componente html. Este seletor de id deve ser aplicado a um unico elemento html, haja vista que o id deve ser unico.Este selertor tem prioridade sobre os demais.

Este seletor semepre começara com uma #, seguido pelo nome do seletor e suas propriedades e valores
