Exemplo 26 CSS Margin

Exemplo CSS Margin

Quando estudamos o BoxModel em CSS aprendemos que todo elemento HTML é representado por uma caixa, e é por meio da margin e do padding que conseguimos definir o posicionamento e alinhamento destes componente. Neste exemplo, estamos criando uma margem que não é vista, porém ela influência diretamente a distância do componente para os outros elementos externos.
