Exemplo 14-html formularios

Os formularios html sao uma das principais formas de se interagir com aplicaçoes web. esses formularios utilizam diversos componentes que melhoraram a experiencia do usuario

Para trabalhar com formularios devemos usar a tag <form>em conjunto com outras tags que vao representar os componentes de formularios tais como labels(rotulos),inputs(caixas de texto),botoes,elementos de checagem,de seleçao de entre outros componenets.Neste exemplo ,demonstraremos o uso dos metodos Get e Post.

Métodos do protocolo http

Get

Quando trabalhamos com a criaçao de formularios html nós deveremos na maior parte das vezes utilizar o protocolo http, esse protocolo possui varios metodos que sao utilizados para facilitar o processo de comunicaçao entre o cliente e o servidor

O metodo get é aquele que anexa os dados do formulario a uma url esses dados do formulario sao visiveis na barra de endereço do navegador.

É importante saber que este metodo nao é recomendado para operaçoes de inclusao pois os dados vao aparecer para qualquer pessoa

Post

Já o metodo post é aquele que envia os dados para o servidor no corpo da pagina, essas informaçoes nao sao visiveis para o cliente sendo um pouco mais seguras que quando utilizamos o metodo get.
