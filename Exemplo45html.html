Exemplo 45 - JavaScript Método Map

Seja um vetor de números pares 0 até 10 [0,2,4,6,8,10]

Vamos criar netes exemplo um novo vetor clicando no botão abaixo e usadno o método Map para multiplicar os elementos por 5 e exibir um novo valor.

Para fazer isso podemos usar a seguinte estrutura:

            for (let i = 0; i < numeros.length; i++) {
                numeros[i] = numeros[i] * 5;
            }

Ou usando Map: 

            const numsMultiplicados = numeros.map(function (numero) {
                return numero * 5;
            });

            ou usando Arrow function
            const numsMultiplicados = numeros.map(numero => numero * 5);
