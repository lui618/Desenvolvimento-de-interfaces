Exemplo 7 html - Listas não Ordenadas <ol> e listas não ordenadas <ul>

Este exemplo mostra como criar listas ordenadas e não ordenadas, incluindo subitens (listas animhadas).

Listas Ordenadas<ol>

As listas ordenadas são aquelas que seguem uma ordem numérica ou por letras. São utilizadas para mostrar conteúdo html em formato de lista

Exemplo de lista ordenada <ol> juntamente com os elementos dessa lista que é representado pela tag <li>

  1. item 1
  2. item 2
  3. item 3
  4. item 4
  5. item 5

Exemoplo 2 de listas ordenadas

  1. item 1
    1. Subitem 1.1
    2. Subitem 1.2
    3. Subitem 1.3
    4. Subitem 1.4
    5. Subitem 1.5
  2. item 2
  3. item 3
  4. item 4
    1. Subitem 4.1
    2. Subitem 4.2
    3. Subitem 4.3
    4. Subitem 4.4
    5. Subitem 4.5
  5. item 5

Listas não ordenadas - <ul>

As listas não ordenadas são aqueles que representam os elementos por meio de bullets. Essas listas são muito utilizadas na criação de menus. Abaixo demonstramos alguns exemplos de listas não ordenadas

Exemplo de lista não ordenadas

  • item 1
  • item 2
    • item 2.1
    • item 2.2
    • item 2.3
      • 2.3.1
      • 2.3.2
      • 2.3.3
  • item 3
  • item 4
  • item 5
