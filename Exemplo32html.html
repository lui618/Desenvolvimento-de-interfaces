Oque é o Grid Layout

O gRid Layout é bidimencional permitindo que o desenvolvedor possa separar o layout da página em linhas e colunas, diferentemente do flex box que é unidimencional ordenando os elemnentos em linhas.

Flexgrid ou Grid Layout

O Flex box e o grid layout pode ser usado em conjunto. Voce pode aplicar o FlexBox a elementos dentro de um layout de grae como Grid Layout aproveitando dos doiis mundos.

A escolha entre flexbox e Grid Layout não se trata de um substiituir o outro, mas sim de entender as necessidades especificas do seu layout e escolher a tecnica que melhor atenda suas necessidades
