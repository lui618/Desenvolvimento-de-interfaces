Exemplo 8 - Uso das tags de imagem <img>, video e iframe

Este exemplo mostra diferentes formas de usar as tags img, video e iframe, incluindo o uso do figure e figcaption para agrupar mídias.

As tags abordadas são:

  • <img>: Tag usada para inserir imagens na página. Essa tag é utilizada em conjunto com os atributos src, alt e width e heigth
  • <video>: Usada para inserir vídeos na página
  • <iframe>: Tag usada para incorporar conteúdo externo como por exemplo vídeos do YouTube
