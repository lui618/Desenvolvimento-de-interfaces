Entendendo os métodos Javascript: Push, Pop, Sort, LastIndexOf, IndexOf

1. "push()"

O método "push()" adiciona um ou mais elementos ao final de um array e retorna o novo comprimento do array.


let frutas = ["maçã", "banana"];
console.log("Array original:", frutas); // Saída: ["maçã", "banana"]

frutas.push("laranja", "uva");
console.log("Array após push:", frutas); // Saída: ["maçã", "banana", "laranja", "uva"]
console.log("Novo comprimento:", frutas.length); // Saída: 4

Neste exemplo, "laranja" e "uva" são adicionados ao final do array "frutas".

Outro exemplo com "push":


let carros = ["Fiat", "Ford"];
carros.push("Chevy", "Honda");
console.log("Array após push:", carros); // Saída: ["Fiat", "Ford", "Chevy", "Honda"]
2. "pop()"

O método "pop()" remove o último elemento de um array e retorna esse elemento removido. Ele altera o comprimento do array.


let carros = ["Fiat", "Ford", "Chevrolet"];
let carroRemovido = carros.pop();
console.log("Array após pop:", carros); // Saída: ["Fiat", "Ford"]
console.log("Carro removido:", carroRemovido); // Saída: "Chevrolet"

Neste exemplo, "Chevrolet" é removido do array "carros" e retornado.

Resultado esperado:
Array original: ["Fiat", "Ford", "Chevrolet"]
Array após pop: ["Fiat", "Ford"]
Elemento removido: Chevrolet
3. "sort()"

O método sort() ordena os elementos de um array no local e retorna o array. A ordenação padrão é alfabética (para strings) ou numérica crescente (para números, se uma função de comparação for fornecida).


let numeros = [5, 2, 8, 1, 9];
console.log("Array original (números):", numeros); // Saída: [5, 2, 8, 1, 9]

numeros.sort(); // Ordenação padrão, pode não funcionar para números como esperado
console.log("Array após sort (padrão):", numeros); // Saída: [1, 2, 5, 8, 9] (neste caso funciona)

let palavras = ["banana", "maçã", "laranja", "abacaxi"];
console.log("Array original (palavras):", palavras); // Saída: ["banana", "maçã", "laranja", "abacaxi"]

palavras.sort();
console.log("Array após sort (palavras):", palavras); // Saída: ["abacaxi", "banana", "laranja", "maçã"]
// Para ordenar números corretamente, precisamos de uma função de comparação: let outrosNumeros = [50, 2, 10, 80, 9]; console.log("Array original (outros números):", outrosNumeros); // Saída: [50, 2, 10, 80, 9] outrosNumeros.sort((a, b) => a - b); // Função de comparação para ordem crescente console.log("Array após sort (numérico):", outrosNumeros); // Saída: [2, 9, 10, 50, 80]

Como (a, b) => a - b para ordem crescente.

Resultado esperado:
Array original (números): [50, 2, 10, 80, 9]
Array após sort (padrão): [10, 2, 50, 80, 9]
Array original (palavras): ["banana", "maçã", "laranja", "abacaxi"]
Array após sort (palavras): ["abacaxi", "banana", "laranja", "maçã"]
Array original (outros números): [50, 2, 10, 80, 9]
Array após sort (numérico): [2, 9, 10, 50, 80]
4. "lastIndexOf()"

O método lastIndexOf() retorna o último índice em que um determinado elemento pode ser encontrado no array, ou -1 se não estiver presente. A busca é feita do final para o início.


let cores = ["vermelho", "verde", "azul", "amarelo"];
console.log("Array:", cores); // Saída: ["vermelho", "verde", "azul", "amarelo"]

let indiceAzul = cores.lastIndexOf("azul");
console.log("Último índice de 'azul':", indiceAzul); // Saída: 2

let indiceRoxo = cores.lastIndexOf("roxo");
console.log("Último índice de 'roxo':", indiceRoxo); // Saída: -1 (não encontrado)

// Procurando a partir de um índice específico (do final para o início)

// Procurando a partir de um índice específico (do final para o início)
let indiceAzulUltimoIndice2 = cores.lastIndexOf("azul", 2);
console.log("Último índice de 'azul' até o índice 2:", indiceAzulUltimoIndice2); // Saída: 2

Essa busca vai do final para o início. Se o elemento não for encontrado, retorna -1.

Resultado esperado:
Array: ["vermelho", "verde", "azul", "amarelo"]
Último índice de 'azul': 2
Último índice de 'roxo': -1
Último índice de 'azul' até o índice 2: 1
5. "indexOf()"

O método indexOf() retorna o primeiro índice em que um determinado elemento pode ser encontrado no array, ou -1 se não estiver presente. A busca é feita do início para o fim.


let animals = ["cachorro", "gato", "pássaro", "gato", "peixe"];
console.log("Array:", animals); // Saída: ["cachorro", "gato", "pássaro", "gato", "peixe"]

let indiceGato = animals.indexOf("gato");
console.log("Primeiro índice de 'gato':", indiceGato); // Saída: 1

let indiceCobra = animals.indexOf("cobra");
console.log("Primeiro índice de 'cobra':", indiceCobra); // Saída: -1 (não encontrado)

// Procurando a partir de um índice específico (do início para o fim)
let indiceGatoAPartirDoIndice2 = animals.indexOf("gato", 2);
console.log("Primeiro índice de 'gato' a partir do índice 2:", indiceGatoAPartirDoIndice2); // Saída: 3

Ele busca do início para o final. Se o elemento não for encontrado, retorna -1.

Resultado esperado:
Array: ["cachorro", "gato", "pássaro", "gato", "peixe"]
Primeiro índice de 'gato': 1
Primeiro índice de 'cobra': -1
Primeiro índice de 'gato' a partir do índice 2: 3
