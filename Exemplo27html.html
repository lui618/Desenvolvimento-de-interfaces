Exemplo 27 - CSS Border e Padding

Exemplo CSS Margin

Quando estudamos o BOXMODEL em css aprendemos que todo elemento HTML é representado por uma caixa, e é por meio da margin e o padding que consiguimos definir o posicionamento e alinhamento destes componentes. Neste exemplo, estamos criando uma margem que não é vista, porem ela influencia diretamente a distância do componente para os outros elementos externos.
